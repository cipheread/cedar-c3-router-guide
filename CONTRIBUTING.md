# 🤝 Contributing

Thank you for helping improve this guide!

## How to Contribute

### Fix a typo or error
1. Click the file on GitHub
2. Click the ✏️ pencil icon
3. Make your edit
4. Click "Propose changes"

### Add carrier APN settings
Edit `docs/connectivity.md` and add your country's carrier APNs to the table.

### Report an issue
Click the **Issues** tab → **New Issue** → describe what's wrong or missing.

## Guidelines
- Keep language clear and simple
- Test any configuration steps before submitting
- One topic per pull request
